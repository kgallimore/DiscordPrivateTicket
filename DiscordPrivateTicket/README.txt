TO GET A TOKEN:
1. Go to https://discordapp.com/developers/applications/
2. Create a new application
3. Create a new bot
4. Copy the token which can be found in the Bot tab under settings, and place it in settings/settings.txt


TO ADD THE BOT TO YOUR SERVER:
You must be on the same application page as the steps to get a token
1. Go to the OAuth2 tab
2. Select 'bot' under scopes, and 'Administrator' under Bot Permissions
3. Visit the generated link in the Scopes section
